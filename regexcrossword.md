# A+
## `+`
- 直前の文字が１文字以上

`A+` → Aが１文字以上

# A|B
## `|`
- 「または」を表す

`A|B` → AまたはB

# A*B?
## `*`
- 直前の文字が０文字以上

## `?`
- 直前の文字が１文字or無し

`A*B?` → Aは０文字以上、Bは１文字or無し

# A?B*
→Aは１文字or０文字、Bは０文字以上

# [AB]+
## `[]`
- []内のいずれかの文字が１字以上

`[AB]+` → AかBのいずれか

# [^AB]
## `^`
- <後ろの文字>以外の任意の文字

`[^AB]` →　AでもBでもない文字
`[^A]` →　A以外の任意の文字
